# 方法选择决策树

面对一道数学建模题，按以下步骤确定建模方法。

## Step 1：判断问题大类

```
题目中是否有数据附件？
├── 有数据（xlsx/csv）→ Step 2A 数据分析分支
└── 无数据/纯文字 → Step 2B 理论建模分支
```

## Step 2A：数据分析分支

```
数据是什么样的？
├── 时间序列（有时间列）→ 预测类
│   ├── 单变量 → ARIMA / Prophet
│   ├── 多变量+非线性 → LSTM / XGBoost
│   └── 需要可解释性 → ARIMA + 特征重要性分析
│
├── 多对象+多指标（需要排名/打分）→ 评价类
│   ├── 指标数据完整 → 熵权法 + TOPSIS（客观）
│   └── 需要专家判断 → AHP + 模糊评价（主客观结合）
│
└── 特征丰富+标签（分类/回归）→ ML类
    ├── 回归 → XGBoost / RandomForest / SVR
    ├── 分类 → XGBoost / LightGBM / SVM
    └── 聚类 → K-Means / DBSCAN / 层次聚类
```

## Step 2B：理论建模分支

```
问题核心是什么？
├── "在约束下最大化/最小化" → 优化类
│   ├── 约束+目标都是线性 → LP (PuLP/ortools)
│   ├── 有0-1/整数变量 → MILP
│   ├── NP-hard/大规模 → GA / PSO / SA
│   └── 多目标 → 加权法 / 帕累托 / NSGA-II
│
├── "节点+边+权重+路径/流" → 图论类
│   ├── 两点间最小成本 → Dijkstra / A*
│   ├── 连通所有节点 → Kruskal / Prim
│   ├── 最大传输量 → Dinic / EK
│   ├── 最短回路 → TSP (GA/SA)
│   └── 车辆调度 → VRP (蚁群/节约算法)
│
├── "物理/生物/经济过程" → 机理分析类
│   ├── 连续变化 → ODE (scipy integrate)
│   ├── 空间+时间 → PDE (有限差分)
│   ├── 随机不确定性 → 蒙特卡洛
│   └── 空间扩散 → 元胞自动机
│
└── "方案比较+多项指标" → 评价类（同Step 2A评价分支）
```

## Step 3：创新加分

选定基本方法后，考虑以下创新方向：

| 创新类型 | 示例 | 难度 |
|---------|------|------|
| **方法改进** | 标准GA加自适应变异率 | ⭐⭐⭐ |
| **组合创新** | 熵权法+CRITIC混合权重→TOPSIS | ⭐⭐ |
| **问题转化** | 将路径规划转化为图着色问题 | ⭐⭐⭐⭐ |
| **参数创新** | 用数据驱动方式确定传统模型的参数 | ⭐⭐ |
| **多方法融合** | ARIMA预测趋势+XGBoost预测残差 | ⭐⭐⭐ |

**选择原则**：选你真正理解并能解释清楚的方向。评委能看出你是否真的懂。
