# 模型库 — 预测类（CO-STAR格式 · DeepSeek V4 Pro）

## 适用场景
- 时间序列预测、回归分析、趋势预测、销量预测
- 核心特征：基于历史数据预测未来，有明确因变量

## 方法速查

| 方法 | 适用 | 数据要求 | 何时选用 |
|------|------|---------|---------|
| **ARIMA** | 单变量时序 | 平稳性检验 | 纯时序，无显著季节性 |
| **Prophet** | 带季节性时序 | 长周期 | 节假日效应+周期性 |
| **LSTM/GRU** | 复杂时序 | 大量数据 | 长依赖非线性 |
| **XGBoost** | 多特征回归 | 特征充分 | 特征多，需要特征重要性 |

## 完整Prompt模板（复制使用）

```
Context:
你正在解决一个数学建模竞赛的预测问题。题目提供了历史数据（含时间序列和/或多维特征）。
你是预测建模专家，需要选出最优模型并给出预测结果。

Objective:
为以下预测问题选择≥3种模型进行对比，选出最优模型并输出预测结果。

Constraints:
- 必须用至少3种不同原理的模型对比（如ARIMA+LSTM+XGBoost）
- 时间序列不能随机划分训练集——按时间顺序
- 评价指标至少3个：MSE, MAE, R²
- 必须做交叉验证（时间序列用滚动窗口）
- 所有模型输出统一格式的预测结果对比表

Output Format:
1. 【数据探索】
   趋势图+周期性分析+相关性热力图
   
2. 【模型对比】
   | 模型 | MSE | MAE | R² | 训练时间 |
   
3. 【最优模型选择】
   选择X，因为Y。参数设置：n_estimators=..., learning_rate=...
   
4. 【预测结果】
   表格 + 预测曲线图（含置信区间）
```

## 常见坑
- 时间序列数据随机shuffle → 泄露未来信息
- LSTM未归一化或预测后忘记反归一化
- XGBoost不调参直接跑 → 过拟合
- 差分阶数未经ADF检验

## 论文写作要点
- 必须有模型对比表（为什么选这个模型）
- 必须展示预测曲线 vs 真实值
- 讨论残差分布（证明模型无偏）
