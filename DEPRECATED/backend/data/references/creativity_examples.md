# 创造性维度三级案例库

> 本文档为审稿 Agent 提供创造性的具体判断标准。每个级别有 3 个真实案例做锚点。
> 审稿 Agent 应将待审论文与这些案例对比，判断其创造性属于哪一级别。

---

## 方法创新（国一级别）

**定义**：提出了新的算法、模型框架、或对标准方法做了实质性改进。

**判断标准**：
- [ ] 方法有明确的"为什么这样做"而非"因为常用"
- [ ] 方法针对本题做了非平凡的适配
- [ ] 方法的改进有可验证的效果（如精度提升、计算加速）

### 案例 1：2021A028 — FAST 主动反射面调节模型

**创新点**：坐标旋转变换降维（三维→准二维）

**为什么是方法创新**：
- 不是直接套用标准优化方法
- 针对 FAST 的几何特性（抛物面+促动器分布）设计了专用的坐标变换
- 将三维优化问题降维为准二维问题，显著降低了计算复杂度
- 结果：接收比 67.46% vs 基准 5.27%（12.8 倍提升）

**与标准方法的区别**：
- 标准方法：直接在三维空间做优化（高维、慢、易陷入局部最优）
- 本文方法：先做坐标变换降维，再在低维空间优化（低维、快、更易找到全局最优）

### 案例 2：2023B477 — 多波束测深合理探测方案

**创新点**：贪心→SA 双方法互验策略

**为什么是方法创新**：
- 不是单一方法求解，而是设计了"双方法互验"机制
- 用模拟退火作为贪心解的"质量证明器"
- 如果 SA 结果与贪心一致 → 高置信度
- 如果不一致 → 说明贪心陷入了局部最优，需要调整

**与标准方法的区别**：
- 标准方法：用一种优化算法求解，假设结果正确
- 本文方法：用两种不同原理的算法互验，提高结果可信度

### 案例 3：2025B157 — SiC 外延层厚度测量

**创新点**：双光束/多光束干涉双路径互验

**为什么是方法创新**：
- 不是单方法求解，而是设计了"双路径自洽验证"机制
- 双光束法得到 d≈5.8μm，多光束法得到 d≈5.7μm
- 两种方法的结果一致性（0.1μm 差异）证明了测量的可靠性

**与标准方法的区别**：
- 标准方法：用一种干涉法测量，假设结果正确
- 本文方法：用两种不同物理原理的干涉法互验，提高测量可信度

## 组合创新（国二级别）

**定义**：将已知方法以非平凡方式组合，解决多步骤问题。

**判断标准**：
- [ ] 组合有逻辑递进（不是随便堆砌）
- [ ] 每步选择有理由
- [ ] 整体效果 > 各部分之和

### 案例 1：2022A001 — 波浪能装置输出功率优化设计

**组合方式**：牛顿力学→RK4→遗传算法

**为什么是组合创新**：
- 三步递进：物理建模（牛顿力学）→ 数值求解（RK4）→ 参数优化（GA）
- 每步选择有理由：牛顿力学描述物理过程，RK4 求解微分方程，GA 搜索最优参数
- 结果：Pmax=229.68W/230.02W/316.58W，量级合理

**与方法创新的区别**：
- 每个方法都是标准方法，没有实质性改进
- 但组合方式有逻辑递进，不是随便堆砌

### 案例 2：2023C228 — 多波束测线布设

**组合方式**：几何→贪心→SA 验证→LS 拟合

**为什么是组合创新**：
- 四步递进：几何分析→贪心求解→SA 验证→LS 拟合
- 每步选择有理由：几何分析建立基础，贪心求解初始方案，SA 验证质量，LS 拟合优化
- 结果：34 条测线/125936m，覆盖率 99.96%

**与方法创新的区别**：
- 没有提出新的算法或框架
- 但组合方式有逻辑递进，且 SA 验证步骤体现了质量意识

### 案例 3：2024B159 — 生产过程次品率检测与拆解决策

**组合方式**：假设检验→DP→贝叶斯→灵敏度

**为什么是组合创新**：
- 四步递进：假设检验（判断是否接收）→ DP（多阶段决策）→ 贝叶斯（处理不确定性）→ 灵敏度（验证鲁棒性）
- 每步选择有理由：假设检验做初步判断，DP 做多阶段优化，贝叶斯处理先验信息，灵敏度验证结果
- 结果：最优利润 66.7 元（确定性）/54.2 元（贝叶斯修正）

**与方法创新的区别**：
- 每个方法都是标准方法，没有实质性改进
- 但组合方式有逻辑递进，且贝叶斯修正步骤体现了对不确定性的处理

## 应用创新（省一级别）

**定义**：将标准方法直接应用于新问题，无实质性改进。

**判断标准**：
- [ ] 方法选择正确但无本题适配
- [ ] 无"为什么选这个方法"的论证
- [ ] 换一个类似问题，方法框架完全不变

### 案例 1：2022C155 — 基于成分数据的古代玻璃制品的成分分析与鉴别

**应用方式**：CLR+聚类+Fisher+相关分析

**为什么是应用创新**：
- 每个方法都是成分数据分析的标准范式
- CLR（中心化对数比变换）是处理成分数据的标准方法
- 聚类+Fisher 判别是分类问题的标准方法
- 相关分析是探索性分析的标准方法
- 没有针对"古代玻璃"这个特殊问题做任何适配

**与组合创新的区别**：
- 方法选择正确，但无"为什么选这个方法"的论证
- 换一个类似的成分数据分析问题（如土壤成分、食品成分），方法框架完全不变

### 案例 2：2021B007 — 乙醇催化裂解制备 C4 烯烃

**应用方式**：多元线性回归+PLS+均匀设计

**为什么是应用创新**：
- 每个方法都是化学工程优化的标准方法
- 多元线性回归是基本的回归方法
- PLS（偏最小二乘）是处理多重共线性的标准方法
- 均匀设计是实验设计的标准方法
- 没有针对"乙醇催化裂解"这个特殊问题做任何适配

**与组合创新的区别**：
- 方法选择正确，但无"为什么选这个方法"的论证
- 换一个类似的化学工程优化问题，方法框架完全不变

### 案例 3：2025C132 — NIPT 检测决策优化

**应用方式**：统计建模+元学习

**为什么是应用创新**：
- 摘要中只有"统计建模"和"元学习"两个笼统词汇
- 无具体方法细节、无数值结果
- 无法判断是否有本题适配
- 从摘要看，这是标准方法的直接应用

**与组合创新的区别**：
- 摘要信息量太低，无法判断是否有逻辑递进
- 无数值结果，无法验证方法的有效性

## 判断流程

审稿 Agent 在评估创造性时，应按以下流程：

1. **识别方法**：论文用了哪些方法？
2. **检查适配**：这些方法是否针对本题做了改进？
3. **对比案例**：与上述 9 个案例对比，最接近哪个？
4. **判定级别**：根据对比结果，判定为方法创新/组合创新/应用创新

**关键问题**：
- 如果把这篇论文的方法框架放到另一个类似问题上，需要改动多少？
  - 改动很大 → 方法创新
  - 改动适中 → 组合创新
  - 几乎不变 → 应用创新
