## 配色方案

### 推荐配色（来自国一/国二论文）

| 论文 | 配色 | 特点 |
|------|------|------|
| 2021A028 (FAST) | 蓝色系为主 + 橙色强调 | 色盲友好，层次分明 |
| 2023B477 (多波束) | 深蓝+浅蓝渐变 + 白色网格 | 专业感强，适合工程图 |
| 2025B157 (SiC) | 红蓝双色对比 | 清晰区分两组数据 |
| 2022A001 (波浪能) | 绿色系 + 灰色背景 | 自然感，适合物理模型 |

### 避免的配色

| 问题 | 案例 | 原因 |
|------|------|------|
| 红绿同时作为主色 | 色盲不友好 | CUMCM 图表禁止项 |
| jet/rainbow colormap | 视觉混乱 | CUMCM 图表禁止项 |
| 纯黑纯白无层次 | 单调 | 评委扫读时不易抓重点 |

## 标注风格

### 轴标签

| 规范 | 示例 | 来源 |
|------|------|------|
| 中文+单位 | "厚度 (μm)" | CUMCM 格式规范 |
| 字号 12pt | 正文字号的一半 | 排版规范 |
| 无缩写 | "反射率" 非 "R" | 评委扫读需要 |

### 图例

| 规范 | 示例 | 来源 |
|------|------|------|
| 位置：右上角或下方 | 不遮挡数据 | 国一论文共性 |
| 边框：无或细线 | 不喧宾夺主 | 国一论文共性 |
| 字号：10pt | 小于轴标签 | 排版规范 |

### 数据标签

| 规范 | 示例 | 来源 |
|------|------|------|
| 关键点标注数值 | 极值、拐点、交点 | 国一论文共性 |
| 格式：保留2-3位小数 | "5.8μm" 非 "5.800000μm" | 简洁原则 |
| 位置：数据点上方 | 不遮挡数据线 | 国一论文共性 |

## 布局模式

### 单图

```
┌─────────────────────────────────┐
│                                 │
│         [图表内容]              │
│                                 │
│  图X：标题（自解释）            │
└─────────────────────────────────┘
宽度：0.8\textwidth
居中
```

### 多图（2×2）

```
┌──────────┐ ┌──────────┐
│  (a) 图1 │ │  (b) 图2 │
└──────────┘ └──────────┘
┌──────────┐ ┌──────────┐
│  (c) 图3 │ │  (d) 图4 │
└──────────┘ └──────────┘
图X：(a) 标题1 (b) 标题2 (c) 标题3 (d) 标题4
```

### 流程图（9框TikZ）

```
┌─────┐   ┌─────┐   ┌─────┐
│ 问题 │ → │ 分析 │ → │ 假设 │
└─────┘   └─────┘   └─────┘
┌─────┐   ┌─────┐   ┌─────┐
│ 模型 │ → │ 求解 │ → │ 验证 │
└─────┘   └─────┘   └─────┘
┌─────┐   ┌─────┐   ┌─────┐
│ 检验 │ → │ 评价 │ → │ 改进 │
└─────┘   └─────┘   └─────┘
每框 ≤7 字
```

## 图表质量检查清单

Agent E 画完图后，用此清单自检：

| # | 检查项 | 通过标准 |
|---|--------|----------|
| 1 | 标题 | 自解释，不含"如图所示" |
| 2 | 轴标签 | 中文+单位，字号12pt |
| 3 | 图例 | 不遮挡数据，字号10pt |
| 4 | 配色 | 色盲友好，无红绿对比 |
| 5 | DPI | ≥300 |
| 6 | 数据标签 | 关键点有数值标注 |
| 7 | 网格线 | 辅助但不干扰 |
| 8 | 字体 | 统一，无乱码 |

## 国一论文图表特征总结

从 15 篇已审论文中观察到的共同特征：

1. **每张图都有明确的"这张图说明了什么"**——不是放上去就完事
2. **关键数值直接标在图上**——不让读者自己去查表
3. **配色统一**——整篇论文用同一套配色方案
4. **图与文字配合**——图旁边有≥2句分析文字
5. **流程图简洁**——每框≤7字，不多不少
