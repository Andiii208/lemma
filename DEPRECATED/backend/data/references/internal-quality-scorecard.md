# Lemma 内部质量评分卡（V8.4）

> **⚠️ 重要声明**：本文档是 Lemma 项目的**内部写作质量自检工具**，非 CUMCM 官方评分标准。  
> CUMCM 官方评奖只使用四项定性原则（`CUMCM-章程-2`），不存在统一的分数分配表。  
> 本评分卡的作用是：写论文时逐项检查，确保质量达标再提交。
>
> **目标**：让你的论文达到「评委拿到不会因为低级问题扣分」的水平。  
> **不保证**：能预测评委最终给什么奖。

## 评分方法

每项检查：**1（最低）→ 5（最高）** 打分。

| 分值 | 含义 |
|------|------|
| 5 | 优秀，无需修改 |
| 4 | 合格，有轻微改进空间 |
| 3 | 有改进空间，但不致命 |
| 2 | 不合格，需修改 |
| 1 | 完全缺失或严重错误 |

### 计算公式

```
维度分 = (Σ该项打分) / (该项数 × 5) × 权重

总分 = Σ 所有维度分
```

**举例**：假设合理性 4 项打了 [4,3,5,4]
```
维度分 = (4+3+5+4) / (4×5) × 20 = 16/20 × 20 = 16 分
```

## 阻断项（一票否决）

以下情况触发 **🔴 需重写**，无论总分多少：

| # | 阻断条件 | 来源 | 说明 |
|---|---------|------|------|
| B1 | 摘要模板化空洞（"本文建立了XX模型，验证了有效性"） | `韩中庚-2017-§4.1` | 换一篇同题论文也能用 |
| B2 | 核心结论数值不可复现 | `格式规范-2025-§5` | 代码跑不出论文里的数 |
| B3 | 模型假设与题目条件严重矛盾 | `CUMCM-章程-2` | 例：连续性假设用于整型决策变量未说明理由 |
| B4 | 论文结构严重残缺（缺模型建立/求解/检验中任意一章） | 国奖论文共性 | **有一问完整缺失该章即触发** |

> ⚠️ 阻断项检查在评分卡之前执行。任何一项触发 → 不评分，直接标注需重写。

## 校准系统变更

### 旧校准（V8.1）

```
目标：Agent 判定 vs "已知等级" → 预测准确率
结果：8/15 = 53%（不靠谱的 ground truth × 不靠谱的预测）
```

### 新校准（V8.4）

```
目标 A（重测信度）：同一篇论文审两次 → 评分差 ≤5 分 → 一致性 ≥80%
目标 B（区分度）：质量明显不同的论文，分数差 ≥15 分
目标 C（阈值验证）：≥85 分的论文，实际评委反馈应为正面的
```

校准不再假装预测 CUMCM 等级，而是验证：
1. **审稿人自己是否稳定**（重测信度）
2. **评分卡能否区分好坏**（区分度）
3. **阈值是否合理**（实际反馈验证）
