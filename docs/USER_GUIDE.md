# Lemma 用户指南

## 快速开始

### 1. 安装

下载对应平台的安装包：
- Windows: `.exe` 安装程序
- macOS: `.dmg` 磁盘映像
- Linux: `.AppImage` 或 `.deb`

### 2. 首次运行引导

首次启动 Lemma 会引导你完成初始设置：

1. **欢迎** — 了解 Lemma 的功能
2. **API Key** — 输入 Anthropic API Key（加密存储）
3. **工作目录** — 选择项目文件夹
4. **预设模板** — 选择适合你需求的预设

### 3. 开始对话

设置完成后即可开始与 Claude 对话。

## 功能说明

### 预设模板

| 预设 | 说明 |
|------|------|
| 📐 数学建模 | 数学建模竞赛全自动求解 |
| 📝 论文写作 | 学术论文撰写助手 |
| 🔬 实验报告 | 实验报告撰写与分析 |
| 📚 文献综述 | 系统性文献检索与综述 |
| 📊 数据挖掘 | 数据探索与模型构建 |

### 文件浏览器

在左侧导航栏点击"文件"可查看工作目录中的文件。

### 成本追踪

点击"成本"可查看当前会话的 Token 消耗和估算成本。

### 对话导出

点击"导出"可将对话导出为 Markdown 文件或复制到剪贴板。

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + N` | 新建会话 |
| `Ctrl/Cmd + O` | 选择工作目录 |
| `Ctrl/Cmd + .` | 停止生成 |
| `Ctrl/Cmd + K` | 清空对话 |
| `Ctrl/Cmd + ,` | 打开设置 |
| `Ctrl/Cmd + Shift + E` | 导出对话 |
| `Shift + Enter` | 消息换行 |

## CLAUDE.md

在工作目录中创建 `CLAUDE.md` 文件可以自定义 Claude 的行为。
Lemma 会在每次对话时自动读取此文件作为系统提示。

## 离线模式

断网时 Lemma 会自动进入离线模式：
- 可以浏览历史会话
- 可以查看文件浏览器
- 无法发送新消息（网络恢复后自动重连）

## 故障排除

### API Key 无效
确保 API Key 格式正确（以 `sk-ant-` 开头），且在 Anthropic 控制台中有效。

### 连接超时
检查网络连接。Lemma 会自动重试，也可以在错误提示中点击"重试"按钮。

### SDK 版本不兼容
如果看到版本警告，请更新 Lemma 到最新版本。
