# 数据可视化规范

## 概述

数据可视化是数据挖掘中探索数据、展示结果的核心手段。规范的可视化能有效传达信息、发现模式、支撑决策。

## 核心概念

### 可视化目的
- **探索性分析**：发现数据分布、异常值、模式
- **结果展示**：展示模型性能、特征重要性、预测结果
- **报告呈现**：出版级图表，清晰传达结论

### 设计原则
- **清晰优先**：信息传达 > 美观
- **少即是多**：避免过度装饰（3D 效果、渐变色、阴影）
- **一致性**：同一报告使用统一配色和风格
- **可访问性**：避免红绿搭配（色盲不友好）

def plot_distribution(df: pd.DataFrame, cols: list, figsize=(15, 4)):
    """绘制数值特征的分布图。"""
    fig, axes = plt.subplots(1, len(cols), figsize=figsize)
    for ax, col in zip(axes, cols):
        df[col].hist(ax=ax, bins=30, edgecolor='black', alpha=0.7)
        ax.axvline(df[col].mean(), color='red', linestyle='--', label=f'Mean: {df[col].mean():.2f}')
        ax.axvline(df[col].median(), color='green', linestyle='--', label=f'Median: {df[col].median():.2f}')
        ax.set_title(col)
        ax.legend(fontsize=8)
    plt.tight_layout()
    plt.show()
```

from sklearn.metrics import roc_curve, precision_recall_curve

def plot_model_evaluation(y_true, y_proba, figsize=(12, 5)):
    """绘制 ROC 曲线和 PR 曲线。"""
    fig, axes = plt.subplots(1, 2, figsize=figsize)

    # ROC 曲线
    fpr, tpr, _ = roc_curve(y_true, y_proba)
    auc_score = roc_auc_score(y_true, y_proba)
    axes[0].plot(fpr, tpr, label=f'AUC = {auc_score:.3f}')
    axes[0].plot([0, 1], [0, 1], 'k--', alpha=0.5)
    axes[0].set_xlabel('False Positive Rate')
    axes[0].set_ylabel('True Positive Rate')
    axes[0].set_title('ROC Curve')
    axes[0].legend()

    # PR 曲线
    precision, recall, _ = precision_recall_curve(y_true, y_proba)
    axes[1].plot(recall, precision)
    axes[1].set_xlabel('Recall')
    axes[1].set_ylabel('Precision')
    axes[1].set_title('Precision-Recall Curve')

    plt.tight_layout()
    plt.show()
```

### 特征重要性图

```python
def plot_feature_importance(model, feature_names: list, top_n: int = 20):
    """绘制特征重要性条形图。"""
    if hasattr(model, 'feature_importances_'):
        importances = model.feature_importances_
    elif hasattr(model, 'coef_'):
        importances = np.abs(model.coef_).flatten()
    else:
        raise ValueError("Model does not support feature importance")

    indices = np.argsort(importances)[-top_n:]
    plt.figure(figsize=(10, 6))
    plt.barh(range(len(indices)), importances[indices])
    plt.yticks(range(len(indices)), [feature_names[i] for i in indices])
    plt.xlabel('Importance')
    plt.title(f'Top {top_n} Feature Importance')
    plt.tight_layout()
    plt.show()
```

## 常见陷阱

| 陷阱 | 说明 | 解决方案 |
|------|------|----------|
| 3D 图表 | 视觉扭曲，难以精确读数 | 使用 2D 图表 + 颜色编码 |
| 截断 Y 轴 | 夸大差异 | Y 轴从 0 开始，或明确标注截断 |
| 过多类别 | 饼图/条形图类别过多难以阅读 | 合并小类别为"其他" |
| 红绿搭配 | 色盲人群无法区分 | 使用蓝橙/蓝红配色 |
| 无标题/标签 | 图表含义不清 | 必须有标题、坐标轴标签、图例 |

## 速查表

| 场景 | 推荐图表 | 关键参数 |
|------|----------|----------|
| 单变量分布 | 直方图 / KDE | `bins=30` |
| 多变量分布 | 小提琴图 / 箱线图 | `sns.violinplot` |
| 相关性 | 热力图 | `sns.heatmap(annot=True)` |
| 类别计数 | 条形图 | `sns.countplot` |
| 散点关系 | 散点图 + 回归线 | `sns.regplot` |
| 时间趋势 | 折线图 | `plt.plot` |
| 模型性能 | ROC / PR 曲线 | `sklearn.metrics` |
| 特征重要性 | 水平条形图 | `plt.barh` |
| 混淆矩阵 | 热力图 | `ConfusionMatrixDisplay` |
| 多模型对比 | 分组条形图 | `plt.bar` + 偏移 |
