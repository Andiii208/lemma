# 数据预处理最佳实践

## 概述

数据预处理是数据挖掘流程中最耗时但最关键的环节。高质量的预处理直接决定模型性能上限。

## 核心概念

### 数据质量维度
- **完整性**：缺失值比例和分布
- **一致性**：同一实体在不同来源的数据是否一致
- **准确性**：数据是否反映真实情况
- **时效性**：数据是否过时

### 缺失值机制
- **MCAR（完全随机缺失）**：缺失与任何变量无关，可直接删除
- **MAR（随机缺失）**：缺失与其他已观测变量有关，用条件填充
- **MNAR（非随机缺失）**：缺失与缺失值本身有关，需领域知识处理

## 代码示例

### 缺失值分析与处理

```python
import pandas as pd
import numpy as np
from sklearn.impute import SimpleImputer, KNNImputer

def analyze_missing(df: pd.DataFrame) -> pd.DataFrame:
    """分析缺失值分布，返回缺失率报告。"""
    missing = df.isnull().sum()
    missing_pct = (missing / len(df) * 100).round(2)
    report = pd.DataFrame({
        'missing_count': missing,
        'missing_pct': missing_pct,
        'dtype': df.dtypes,
    })
    return report[report['missing_count'] > 0].sort_values('missing_pct', ascending=False)

def impute_numeric(df: pd.DataFrame, col: str, strategy: str = 'auto') -> pd.DataFrame:
    """智能填充数值列缺失值。auto 模式根据偏度选择策略。"""
    if strategy == 'auto':
        skewness = df[col].skew()
        strategy = 'median' if abs(skewness) > 1 else 'mean'
    imputer = SimpleImputer(strategy=strategy)
    df[col] = imputer.fit_transform(df[[col]])
    return df
```

### 异常值检测与处理

```python
from scipy import stats

def detect_outliers_iqr(series: pd.Series, factor: float = 1.5) -> pd.Series:
    """IQR 方法检测异常值，返回布尔掩码。"""
    q1, q3 = series.quantile(0.25), series.quantile(0.75)
    iqr = q3 - q1
    lower, upper = q1 - factor * iqr, q3 + factor * iqr
    return (series < lower) | (series > upper)

def winsorize_series(series: pd.Series, limits: tuple = (0.05, 0.05)) -> pd.Series:
    """Winsorize 截断异常值到指定分位数范围。"""
    from scipy.stats import mstats
    return pd.Series(mstats.winsorize(series, limits=limits), index=series.index)
```

### 数据类型优化

```python
def optimize_dtypes(df: pd.DataFrame) -> pd.DataFrame:
    """自动优化 DataFrame 的数据类型以节省内存。"""
    for col in df.select_dtypes(include=['int64']).columns:
        df[col] = pd.to_numeric(df[col], downcast='integer')
    for col in df.select_dtypes(include=['float64']).columns:
        df[col] = pd.to_numeric(df[col], downcast='float')
    for col in df.select_dtypes(include=['object']).columns:
        if df[col].nunique() / len(df) < 0.5:
            df[col] = df[col].astype('category')
    return df
```

## 常见陷阱

| 陷阱 | 说明 | 解决方案 |
|------|------|----------|
| 全局填充 | 用整个数据集的统计量填充训练集 | 在交叉验证循环内填充 |
| 删除缺失行 | 丢失大量信息 | 分析缺失机制后选择填充策略 |
| 忽略类别编码 | 测试集出现新类别导致报错 | 使用 `handle_unknown='ignore'` |
| 忘记逆变换 | 预测结果未还原缩放 | 保存 scaler 并调用 `inverse_transform` |

## 速查表

| 场景 | 推荐方法 | sklearn 实现 |
|------|----------|-------------|
| 数值缺失（正态） | 均值填充 | `SimpleImputer(strategy='mean')` |
| 数值缺失（偏态） | 中位数填充 | `SimpleImputer(strategy='median')` |
| 类别缺失 | 众数填充 | `SimpleImputer(strategy='most_frequent')` |
| 复杂缺失 | KNN 填充 | `KNNImputer(n_neighbors=5)` |
| 异常值截断 | Winsorize | `scipy.stats.mstats.winsorize` |
| 内存优化 | 类型降级 | `pd.to_numeric(downcast='integer')` |
