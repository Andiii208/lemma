# 模型评估与调参

## 概述

模型评估是验证模型泛化能力的关键环节。正确的评估策略能避免过拟合和选择偏差，确保模型在实际应用中的表现。

## 核心概念

### 评估指标体系

#### 分类指标
- **Accuracy**：整体正确率，类别不平衡时不可靠
- **Precision**：查准率，预测为正的样本中真正为正的比例
- **Recall**：查全率，真正为正的样本中被正确预测的比例
- **F1 Score**：Precision 和 Recall 的调和平均
- **AUC-ROC**：不同阈值下的综合表现，不受类别比例影响

#### 回归指标
- **MSE / RMSE**：均方误差，对大误差敏感
- **MAE**：平均绝对误差，对异常值鲁棒
- **R²**：解释方差比例，1 为完美拟合
- **MAPE**：百分比误差，适合比较不同量级的目标

### 交叉验证策略
- **K-Fold**：标准方法，数据量充足时使用
- **StratifiedKFold**：保持每折类别比例一致
- **TimeSeriesSplit**：时间序列数据，只能用过去预测未来
- **GroupKFold**：同一实体的数据不同时出现在训练集和验证集

## 代码示例

### 完整评估流程

```python
from sklearn.model_selection import cross_val_score, StratifiedKFold
from sklearn.metrics import classification_report, roc_auc_score
import numpy as np

def evaluate_model(model, X_train, y_train, X_test, y_test, cv=5):
    """完整评估流程：交叉验证 + 测试集评估。"""
    # 交叉验证
    skf = StratifiedKFold(n_splits=cv, shuffle=True, random_state=42)
    cv_scores = cross_val_score(model, X_train, y_train, cv=skf, scoring='f1')

    # 训练并预测
    model.fit(X_train, y_train)
    y_pred = model.predict(X_test)
    y_proba = model.predict_proba(X_test)[:, 1] if hasattr(model, 'predict_proba') else None

    # 报告
    print(f"CV F1: {cv_scores.mean():.4f} +/- {cv_scores.std():.4f}")
    print(classification_report(y_test, y_pred))
    if y_proba is not None:
        print(f"AUC-ROC: {roc_auc_score(y_test, y_proba):.4f}")

    return cv_scores
```

### 超参数调优

```python
from sklearn.model_selection import RandomizedSearchCV
from scipy.stats import uniform, randint

def tune_hyperparameters(model, param_dist, X_train, y_train,
                         n_iter=50, cv=5, scoring='f1'):
    """随机搜索超参数调优。"""
    search = RandomizedSearchCV(
        model, param_dist, n_iter=n_iter, cv=cv,
        scoring=scoring, random_state=42, n_jobs=-1,
    )
    search.fit(X_train, y_train)
    print(f"Best score: {search.best_score_:.4f}")
    print(f"Best params: {search.best_params_}")
    return search.best_estimator_, search.best_params_
```

### 学习曲线诊断

```python
from sklearn.model_selection import learning_curve
import matplotlib.pyplot as plt

def plot_learning_curve(model, X, y, cv=5):
    """绘制学习曲线，诊断过拟合/欠拟合。"""
    train_sizes, train_scores, val_scores = learning_curve(
        model, X, y, cv=cv, train_sizes=np.linspace(0.1, 1.0, 10),
        scoring='f1', n_jobs=-1,
    )
    plt.figure(figsize=(10, 6))
    plt.plot(train_sizes, train_scores.mean(axis=1), label='Training')
    plt.plot(train_sizes, val_scores.mean(axis=1), label='Validation')
    plt.fill_between(train_sizes,
                     train_scores.mean(axis=1) - train_scores.std(axis=1),
                     train_scores.mean(axis=1) + train_scores.std(axis=1), alpha=0.1)
    plt.fill_between(train_sizes,
                     val_scores.mean(axis=1) - val_scores.std(axis=1),
                     val_scores.mean(axis=1) + val_scores.std(axis=1), alpha=0.1)
    plt.xlabel('Training Size')
    plt.ylabel('F1 Score')
    plt.title('Learning Curve')
    plt.legend()
    plt.grid(True)
    plt.show()
```

### 混淆矩阵分析

```python
from sklearn.metrics import confusion_matrix, ConfusionMatrixDisplay

def plot_confusion_matrix(y_true, y_pred, labels=None):
    """绘制混淆矩阵。"""
    cm = confusion_matrix(y_true, y_pred)
    disp = ConfusionMatrixDisplay(cm, display_labels=labels)
    disp.plot(cmap='Blues', values_format='d')
    plt.title('Confusion Matrix')
    plt.show()
```

## 常见陷阱

| 陷阱 | 说明 | 解决方案 |
|------|------|----------|
| 用 Accuracy 评估不平衡数据 | 99% 负样本时全部预测负也有 99% 准确率 | 用 F1 / AUC |
| 单次随机划分 | 结果受划分影响大 | 交叉验证 + 多次重复 |
| 调参用测试集 | 测试集信息泄露 | 独立验证集或交叉验证 |
| 忽略业务约束 | 模型在技术指标上好但业务不可用 | 结合业务需求选择指标 |
| 不报告不确定性 | 只报告均值，忽略方差 | 报告 mean +/- std |

## 速查表

| 场景 | 推荐指标 | 推荐 CV 策略 |
|------|----------|-------------|
| 二分类（均衡） | F1 / AUC | StratifiedKFold(5) |
| 二分类（不均衡） | F1 / PR-AUC | StratifiedKFold(5) + class_weight |
| 多分类 | Macro-F1 | StratifiedKFold(5) |
| 回归 | RMSE / R² | KFold(5) |
| 时间序列 | MAE / MAPE | TimeSeriesSplit(5) |
| 大数据集 | AUC | KFold(3) 节省时间 |
| 小数据集 | LOO / F1 | LeaveOneOut / RepeatedKFold |
