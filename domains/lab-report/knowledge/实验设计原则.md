# 实验设计原则

## 变量类型
- **自变量**: 研究者操控的变量
- **因变量**: 被测量的结果变量
- **控制变量**: 保持不变的变量

## 样本量计算
- 统计功效 (1-β) ≥ 0.80
- 显著性水平 α = 0.05
- 效应量：小(0.2)、中(0.5)、大(0.8)

## 统计检验选择

| 场景 | 方法 |
|------|------|
| 两组均值 | t 检验 |
| 多组均值 | ANOVA |
| 非正态 | Mann-Whitney U |
| 分类变量 | 卡方检验 |

## 效应量
- Cohen's d: 小(0.2), 中(0.5), 大(0.8)
- 报告格式: t(28) = 2.45, p = .021, d = 0.89
