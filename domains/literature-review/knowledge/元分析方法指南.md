# 元分析方法指南

## 元分析基本原理

元分析（Meta-Analysis）是一种定量合并多个独立研究结果的统计方法。其核心思想是：将多个研究的效应量加权平均，得到一个更精确、更可靠的综合效应估计。

### 何时使用元分析

- 多个独立研究检验同一假设，但结论不一致
- 单个研究样本量不足，统计效力低
- 需要量化效应大小，而非仅判断"是否显著"
- 系统综述的定量部分

### 元分析的基本流程

1. 明确研究问题（PICO 框架）
2. 系统检索文献
3. 筛选纳入研究（PRISMA 流程图）
4. 提取数据（效应量、样本量、置信区间）
5. 选择模型（固定效应 / 随机效应）
6. 计算合并效应量
7. 异质性检验
8. 发表偏倚检测
9. 敏感性分析

## 效应量计算

### Cohen's d（连续变量）

用于比较两组均值差异：

$$d = \frac{\bar{X}_1 - \bar{X}_2}{S_p}$$

其中合并标准差：

$$S_p = \sqrt{\frac{(n_1 - 1)S_1^2 + (n_2 - 1)S_2^2}{n_1 + n_2 - 2}}$$

小样本校正（Hedges' g）：

$$g = d \times \left(1 - \frac{3}{4(n_1 + n_2 - 2) - 1}\right)$$

效应量解读：

| Cohen's d | 效应大小 |
|-----------|---------|
| 0.2 | 小效应 |
| 0.5 | 中等效应 |
| 0.8 | 大效应 |

### 比值比（Odds Ratio, OR）

用于二分类数据：

$$OR = \frac{a \times d}{b \times c}$$

| | 事件发生 | 事件未发生 |
|---|---------|-----------|
| 实验组 | a | b |
| 对照组 | c | d |

对数比值比的方差：

$$\text{Var}(\ln OR) = \frac{1}{a} + \frac{1}{b} + \frac{1}{c} + \frac{1}{d}$$

### 相对危险度（Risk Ratio, RR）

$$RR = \frac{a / (a+b)}{c / (c+d)}$$

对数相对危险度的方差：

$$\text{Var}(\ln RR) = \frac{1}{a} - \frac{1}{a+b} + \frac{1}{c} - \frac{1}{c+d}$$

### 相关系数 r

Fisher's Z 变换后进行元分析：

$$Z = \frac{1}{2} \ln\left(\frac{1+r}{1-r}\right)$$

方差：$\text{Var}(Z) = \frac{1}{n-3}$

合并后再反变换：$r = \frac{e^{2Z} - 1}{e^{2Z} + 1}$

## 固定效应 vs 随机效应模型

### 固定效应模型（Fixed-Effect Model）

假设所有研究估计的是同一个真实效应量 $\theta$，研究间的差异仅来自抽样误差。

合并效应量：

$$\hat{\theta} = \frac{\sum w_i \theta_i}{\sum w_i}$$

权重（逆方差法）：$w_i = \frac{1}{\text{Var}(\theta_i)}$

适用场景：纳入研究同质性高（如同一实验条件下的重复实验）

### 随机效应模型（Random-Effects Model）

假设每个研究有其自己的真实效应量 $\theta_i$，$\theta_i \sim N(\mu, \tau^2)$。

权重：$w_i^* = \frac{1}{\text{Var}(\theta_i) + \hat{\tau}^2}$

其中 $\hat{\tau}^2$ 为研究间方差的估计（通常用 DerSimonian-Laird 方法）：

$$\hat{\tau}^2 = \frac{Q - (k-1)}{C}$$

$$Q = \sum w_i (\theta_i - \hat{\theta})^2, \quad C = \sum w_i - \frac{\sum w_i^2}{\sum w_i}$$

适用场景：纳入研究存在异质性（如不同实验条件、不同人群）

### 模型选择建议

| 条件 | 推荐模型 |
|------|---------|
| $I^2 < 50\%$ 且 Q 检验不显著 | 固定效应 |
| $I^2 \geq 50\%$ 或 Q 检验显著 | 随机效应 |
| 纳入研究高度同质 | 固定效应 |
| 目标是推广到更广泛的情境 | 随机效应 |
| 不确定时 | 随机效应（更保守） |

## 异质性检验

### Q 统计量（Cochran's Q 检验）

$$Q = \sum_{i=1}^{k} w_i (\theta_i - \hat{\theta})^2$$

- $H_0$：所有研究的真实效应量相同
- Q 服从自由度为 $k-1$ 的卡方分布
- $p < 0.05$ 表示存在显著异质性

### I² 统计量

$$I^2 = \frac{Q - (k-1)}{Q} \times 100\%$$

| $I^2$ | 异质性程度 |
|-------|-----------|
| 0-25% | 低异质性 |
| 25-50% | 中等异质性 |
| 50-75% | 高异质性 |
| 75-100% | 极高异质性 |

### $\tau^2$ 和 $\tau$

- $\tau^2$：研究间方差，反映真实效应量的变异程度
- $\tau$：研究间标准差，与效应量同量纲

## 森林图解读

森林图（Forest Plot）是元分析的核心可视化，展示每个研究的效应量及其置信区间。

```
研究           效应量 [95% CI]        权重
─────────────────────────────────────────
Study 1        □───────               15.2%
Study 2           □──────             12.8%
Study 3         □──────               18.5%
Study 4              □─────           10.3%
Study 5        □───────               14.7%
Study 6           □──────             11.2%
Study 7        □───────               17.3%
─────────────────────────────────────────
合并效应量     ◆─────                 100%
─────────────────────────────────────────
               ←对照更好  实验更好→
```

解读要点：
- 每个方块代表一个研究的点估计，大小代表权重
- 横线代表 95% 置信区间
- 菱形代表合并效应量及其置信区间
- 竖线（无效线）：OR/RR 为 1，Cohen's d 为 0

## 发表偏倚检测

### 漏斗图（Funnel Plot）

横轴为效应量，纵轴为标准误（或样本量）。对称的漏斗图表示无发表偏倚。

```
标准误（大）
    |         *
    |       *   *
    |     *   .   *
    |   *   .   .   *
    | *   .   .   .   *
    |*   .   .   .   .   *
    └───────────────────────── 效应量
```

不对称的表现：
- 右下角缺失（小样本阴性结果未发表）
- 左下角缺失（小样本阳性结果过度发表）

### Egger 回归检验

将效应量除以标准误作为因变量，标准误的倒数作为自变量进行回归：

$$\frac{\theta_i}{SE_i} = \alpha + \beta \times \frac{1}{SE_i} + \epsilon_i$$

- $H_0: \alpha = 0$（截距为 0，无偏倚）
- $p < 0.05$ 表示存在显著发表偏倚

### Begg 秩相关检验

检验效应量与方差的秩相关性。$p < 0.05$ 表示存在偏倚。

### 剪补法（Trim and Fill）

估计缺失研究的数量和位置，填补后重新计算合并效应量。

## Python 元分析代码

### 基础元分析（statsmodels）

```python
import numpy as np
from scipy import stats

def fixed_effect_meta(effects, variances):
    """固定效应模型元分析（逆方差法）"""
    weights = 1.0 / variances
    pooled_effect = np.sum(weights * effects) / np.sum(weights)
    pooled_se = np.sqrt(1.0 / np.sum(weights))
    ci_lower = pooled_effect - 1.96 * pooled_se
    ci_upper = pooled_effect + 1.96 * pooled_se
    z = pooled_effect / pooled_se
    p_value = 2 * (1 - stats.norm.cdf(abs(z)))
    return {
        'effect': pooled_effect,
        'se': pooled_se,
        'ci': (ci_lower, ci_upper),
        'z': z,
        'p': p_value
    }

def random_effects_meta(effects, variances):
    """随机效应模型元分析（DerSimonian-Laird）"""
    k = len(effects)
    weights_fe = 1.0 / variances
    pooled_fe = np.sum(weights_fe * effects) / np.sum(weights_fe)

    # Q 统计量
    Q = np.sum(weights_fe * (effects - pooled_fe)**2)
    df = k - 1
    p_heterogeneity = 1 - stats.chi2.cdf(Q, df)

    # I² 统计量
    I2 = max(0, (Q - df) / Q * 100)

    # tau² 估计
    C = np.sum(weights_fe) - np.sum(weights_fe**2) / np.sum(weights_fe)
    tau2 = max(0, (Q - df) / C)

    # 随机效应权重
    weights_re = 1.0 / (variances + tau2)
    pooled_effect = np.sum(weights_re * effects) / np.sum(weights_re)
    pooled_se = np.sqrt(1.0 / np.sum(weights_re))
    ci_lower = pooled_effect - 1.96 * pooled_se
    ci_upper = pooled_effect + 1.96 * pooled_se
    z = pooled_effect / pooled_se
    p_value = 2 * (1 - stats.norm.cdf(abs(z)))

    return {
        'effect': pooled_effect,
        'se': pooled_se,
        'ci': (ci_lower, ci_upper),
        'z': z,
        'p': p_value,
        'Q': Q,
        'p_heterogeneity': p_heterogeneity,
        'I2': I2,
        'tau2': tau2
    }

# 示例数据：7 个研究的 Cohen's d
effects = np.array([0.35, 0.52, 0.28, 0.61, 0.45, 0.38, 0.55])
sample_sizes = np.array([50, 80, 40, 100, 60, 45, 70])
variances = 4/sample_sizes + effects**2/(2*sample_sizes)  # 近似方差

print("=== 固定效应模型 ===")
fe_result = fixed_effect_meta(effects, variances)
print(f"合并效应量: {fe_result['effect']:.3f}")
print(f"95% CI: [{fe_result['ci'][0]:.3f}, {fe_result['ci'][1]:.3f}]")
print(f"p = {fe_result['p']:.4f}")

print("\n=== 随机效应模型 ===")
re_result = random_effects_meta(effects, variances)
print(f"合并效应量: {re_result['effect']:.3f}")
print(f"95% CI: [{re_result['ci'][0]:.3f}, {re_result['ci'][1]:.3f}]")
print(f"p = {re_result['p']:.4f}")
print(f"Q = {re_result['Q']:.2f}, p = {re_result['p_heterogeneity']:.4f}")
print(f"I² = {re_result['I2']:.1f}%")
print(f"τ² = {re_result['tau2']:.4f}")
```

### 效应量计算函数

```python
def cohens_d(mean1, sd1, n1, mean2, sd2, n2):
    """计算 Cohen's d 和 Hedges' g"""
    sp = np.sqrt(((n1-1)*sd1**2 + (n2-1)*sd2**2) / (n1+n2-2))
    d = (mean1 - mean2) / sp
    # Hedges' g 校正
    df = n1 + n2 - 2
    correction = 1 - 3 / (4 * df - 1)
    g = d * correction
    # 方差
    var_d = (n1+n2)/(n1*n2) + d**2/(2*(n1+n2))
    return {'d': d, 'g': g, 'var_d': var_d}

def odds_ratio(a, b, c, d):
    """计算比值比（OR）"""
    OR = (a * d) / (b * c)
    ln_OR = np.log(OR)
    var_lnOR = 1/a + 1/b + 1/c + 1/d
    se_lnOR = np.sqrt(var_lnOR)
    ci_lower = np.exp(ln_OR - 1.96 * se_lnOR)
    ci_upper = np.exp(ln_OR + 1.96 * se_lnOR)
    return {'OR': OR, 'ln_OR': ln_OR, 'var_lnOR': var_lnOR,
            'ci': (ci_lower, ci_upper)}

def risk_ratio(a, b, c, d):
    """计算相对危险度（RR）"""
    RR = (a/(a+b)) / (c/(c+d))
    ln_RR = np.log(RR)
    var_lnRR = 1/a - 1/(a+b) + 1/c - 1/(c+d)
    se_lnRR = np.sqrt(var_lnRR)
    ci_lower = np.exp(ln_RR - 1.96 * se_lnRR)
    ci_upper = np.exp(ln_RR + 1.96 * se_lnRR)
    return {'RR': RR, 'ln_RR': ln_RR, 'var_lnRR': var_lnRR,
            'ci': (ci_lower, ci_upper)}
```

### Egger 检验和森林图

```python
def egger_test(effects, variances):
    """Egger 回归检验发表偏倚"""
    se = np.sqrt(variances)
    precision = 1.0 / se
    std_effect = effects / se

    # 线性回归：std_effect = alpha + beta * precision
    from scipy.stats import linregress
    slope, intercept, r_value, p_value, std_err = linregress(precision, std_effect)

    return {
        'intercept': intercept,
        'p_value': p_value,
        'significant': p_value < 0.05
    }

def plot_forest(effects, variances, study_names=None):
    """绘制森林图"""
    import matplotlib.pyplot as plt

    k = len(effects)
    if study_names is None:
        study_names = [f'Study {i+1}' for i in range(k)]

    se = np.sqrt(variances)
    ci_lower = effects - 1.96 * se
    ci_upper = effects + 1.96 * se

    # 合并效应量
    result = random_effects_meta(effects, variances)

    fig, ax = plt.subplots(figsize=(10, k*0.5 + 2))

    y_pos = np.arange(k, 0, -1)

    # 绘制各研究
    for i in range(k):
        ax.plot([ci_lower[i], ci_upper[i]], [y_pos[i], y_pos[i]], 'b-', linewidth=1)
        ax.plot(effects[i], y_pos[i], 'bs', markersize=8)

    # 绘制合并效应量
    ax.axvline(x=result['effect'], color='r', linestyle='--', alpha=0.5)
    ax.plot(result['effect'], 0, 'rD', markersize=10)
    ax.plot([result['ci'][0], result['ci'][1]], [0, 0], 'r-', linewidth=2)

    # 无效线
    ax.axvline(x=0, color='k', linestyle=':', alpha=0.5)

    ax.set_yticks(list(y_pos) + [0])
    ax.set_yticklabels(study_names + ['Pooled'])
    ax.set_xlabel('Effect Size (Cohen\'s d)')
    ax.set_title('Forest Plot')
    plt.tight_layout()
    plt.savefig('forest_plot.png', dpi=150)
    plt.show()
```

## 常见陷阱

### 纳入标准不严格

- 纳入质量低的研究会降低合并结果的可靠性
- 应进行质量评估（如 Newcastle-Ottawa 量表、Cochrane 偏倚风险评估）

### 忽略异质性

- 异质性高时使用固定效应模型会导致错误结论
- 应报告 $I^2$ 和 $\tau^2$，高异质性时使用随机效应模型

### 发表偏倚未检测

- 阴性结果的研究更难发表，导致效应量被高估
- 必须绘制漏斗图、进行 Egger/Begg 检验

### 双重数据提取

- 同一研究的多个报告被重复纳入
- 应仔细检查研究的作者、样本量、发表时间

### 生态学谬误

- 将群体层面的结论推广到个体层面
- 元分析的结论应限于纳入研究的总体

### 统计效力不足

- 纳入研究数量太少时，异质性检验和发表偏倚检测的统计效力低
- Q 检验在研究数 < 5 时几乎无法检测异质性

### 效应量转换不当

- 不同研究报告的指标不同（均值差、OR、RR 等），需要统一转换
- 转换公式使用不当会引入误差

### 亚组分析的多重比较

- 进行大量亚组分析会增加假阳性风险
- 应预先设定亚组分析计划，或使用 Bonferroni 校正
