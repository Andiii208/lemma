# 系统综述写作指南

## 系统综述 vs 叙述性综述

| 特征 | 系统综述 | 叙述性综述 |
|------|---------|-----------|
| 方法 | 系统化、可复现 | 非系统化 |
| 检索 | 全面、有策略 | 选择性 |
| 筛选 | 有明确标准 | 主观判断 |
| 评估 | 质量评估 | 通常无 |
| 综合 | 定量/定性综合 | 定性总结 |

## PRISMA 流程

### 1. 识别（Identification）
- 数据库检索
- 其他来源（手动检索、专家推荐）

### 2. 筛选（Screening）
- 标题/摘要筛选
- 全文筛选
- 记录排除原因

### 3. 资格评估（Eligibility）
- 应用纳入/排除标准
- 质量评估

### 4. 包括（Included）
- 最终纳入文献
- Meta 分析（如适用）

## 数据提取表

| 字段 | 内容 |
|------|------|
| 作者 | 第一作者 |
| 年份 | 发表年份 |
| 国家 | 研究国家 |
| 样本量 | N = ? |
| 方法 | 研究设计 |
| 结果 | 主要发现 |
| 质量评分 | 高/中/低 |

## Meta 分析基础

### 异质性检验
- **I² 统计量**：异质性程度
  - <25%：低异质性
  - 25-75%：中等异质性
  - >75%：高异质性

### 效应量合并
- 固定效应模型：假设效应量相同
- 随机效应模型：假设效应量不同

### 森林图
- 展示各研究效应量
- 显示置信区间
- 显示合并效应量

## 写作模板

```markdown
# [主题] 的系统综述

## 摘要
背景 | 目的 | 方法 | 结果 | 结论

## 1. 引言
- 研究背景
- 综述目的
- 研究问题

## 2. 方法
- 检索策略
- 纳入/排除标准
- 数据提取
- 质量评估
- 数据综合

## 3. 结果
- 检索结果（PRISMA 流程图）
- 纳入研究特征
- 质量评估结果
- 综合结果

## 4. 讨论
- 主要发现
- 局限性
- 未来方向

## 5. 结论
```
