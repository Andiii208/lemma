# 灵敏度分析与模型检验

## 灵敏度分析

### 目的
- 评估模型对参数变化的敏感程度
- 识别关键参数
- 验证模型的稳健性

### 方法

#### 单因素灵敏度分析
- 固定其他参数，变化一个参数
- 观察输出变化
- 计算灵敏度指数

```python
# 单因素灵敏度分析示例
def sensitivity_analysis(model, base_params, param_name, range_pct=0.2):
    """对指定参数进行灵敏度分析"""
    results = []
    base_value = base_params[param_name]
    
    for pct in np.linspace(-range_pct, range_pct, 11):
        params = base_params.copy()
        params[param_name] = base_value * (1 + pct)
        output = model(**params)
        results.append({
            'param_change': pct,
            'output': output,
            'param_name': param_name
        })
    
    return results
```

#### 全局灵敏度分析
- **Sobol 指数**：一阶和总效应指数
- **Morris 方法**：筛选重要参数
- **FAST 方法**：傅里叶振幅检验

### 报告格式
- 灵敏度指数表
- 龙卷风图（Tornado Chart）
- 参数扫描图

## 模型检验

### 残差分析
- 残差正态性检验（Shapiro-Wilk）
- 残差独立性检验（Durbin-Watson）
- 残差方差齐性检验

### 交叉验证
- **K 折交叉验证**：数据分为 K 份
- **留一法**：每次留一个样本
- **时间序列交叉验证**：按时间划分

### 拟合优度
- **R²**：决定系数
- **RMSE**：均方根误差
- **MAE**：平均绝对误差
- **AIC/BIC**：信息准则

```python
from sklearn.metrics import r2_score, mean_squared_error, mean_absolute_error

r2 = r2_score(y_true, y_pred)
rmse = np.sqrt(mean_squared_error(y_true, y_pred))
mae = mean_absolute_error(y_true, y_pred)
```
