# 学术论文审稿回复技巧

## 回复原则

1. **尊重审稿人**：即使不同意也要礼貌
2. **逐条回复**：每个意见都要回应
3. **具体改进**：说明实际做了什么修改
4. **引用位置**：指出修改在原文的位置

## 回复格式

```
Reviewer 1, Comment 1:
[审稿人意见原文]

Response:
感谢审稿人的建议。我们已经[具体改进措施]。
修改位置：Section X, Page Y, Line Z。
[如有必要，引用修改后的原文]
```

## 常见情况处理

### 同意并修改
```
感谢审稿人指出这个问题。我们已经[具体修改]。
修改位置：Section 3.2, Page 7。
```

### 部分同意
```
感谢审稿人的建议。我们同意[同意部分]，但对于[不同意部分]，原因是[理由]。
我们已经在文中添加了[补充说明]来解释这一点。
```

### 不同意但尊重
```
感谢审稿人的深入思考。这个问题确实值得讨论。
然而，由于[具体原因]，我们选择保持当前方法。
我们在 Discussion 中添加了相关讨论（Section 5, Page 12）。
```

## 常见错误

1. ❌ 只说"已修改"而不说明具体改了什么
2. ❌ 对审稿人意见置之不理
3. ❌ 回复过于冗长
4. ❌ 态度防御性或傲慢
